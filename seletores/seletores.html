Agrupamento de elementos

Os elementos <div> e <span> são contêineres genéricos utilizados para agrupar e estruturar partes de um documento, sendo frequentemente associados aos atributos class e id para fins de estilização e manipulação via script. A <div> é um elemento de nível de bloco (block-level), que inicia uma nova linha e ocupa toda a largura disponível, ideal para organizar grandes seções de layout. Por outro lado, o <span> é um elemento em linha (inline), aplicado para envolver pequenas porções de texto ou elementos dentro de um parágrafo sem interromper o fluxo do conteúdo.

Elementos Block-level

O elemento <div> é classificado como um elemento de nível de bloco (block-level), funcionando como um contêiner genérico para agrupar outros elementos HTML e estruturar seções do documento. Por padrão, ele sempre inicia em uma nova linha e expande sua largura para ocupar todo o espaço horizontal disponível do seu elemento pai, sendo essencial para a criação de layouts e a aplicação de estilos específicos a blocos de conteúdo distintos.

Elementos Inline

O elemento <span> é classificado como um elemento em linha (inline), funcionando como um contêiner genérico e semântico neutro que permite marcar e isolar trechos específicos de conteúdo dentro de uma mesma linha ou parágrafo. Diferente dos elementos de bloco, o <span> não interrompe o fluxo do texto nem inicia uma nova linha, sendo amplamente utilizado em conjunto com o CSS para aplicar estilos personalizados, como cores ou fontes diferenciadas, a palavras ou frases isoladas no corpo de um documento HTML.

Seletores Personalizados

Os seletores de classe e de ID possibilitam a personalização de elementos HTML específicos para fins de estilização via CSS, permitindo aplicar regras de design de forma direcionada. O seletor de classe, identificado pelo símbolo de ponto ".", é ideal para aplicar o mesmo estilo a múltiplos elementos em uma página, enquanto o seletor de ID, identificado pela cerquilha (#), deve ser único por documento, servindo para estilizar um elemento exclusivo e possuindo maior prioridade na cascata do CSS.

Seletor do tipo id

O seletor de ID utiliza o atributo id de um elemento HTML para selecionar e aplicar estilos a um único elemento específico dentro da página, garantindo que cada identificador seja exclusivo no documento. No CSS, esse seletor é representado pelo caractere "#", possuindo uma especificidade maior do que os seletores de tag ou de classe, o que assegura que suas regras prevaleçam em casos de conflitos de estilização para aquele elemento determinado.

Seletor do tipo classe

O seletor de classe é utilizado para selecionar e estilizar elementos HTML que possuem um atributo class específico, sendo representado no CSS pelo caractere de ponto ".". Diferente do ID, a classe permite que o mesmo identificador seja aplicado a múltiplos elementos dentro de uma única página, tornando-a a ferramenta ideal para criar padrões de design reutilizáveis e agrupar componentes que devem compartilhar as mesmas características visuais.

Pseudo-Classes

As pseudoclasses são utilizadas para definir um estado especial de um elemento, sendo representadas pelo símbolo de dois pontos (:). Elas estão relacionadas a um elemento ou a uma classe, permitindo aplicar estilos em situações específicas, como quando o usuário passa o mouse sobre um item ou quando um link já foi visitado.

Pseudo-Elementos

Os pseudoelementos são utilizados para estilizar partes específicas de um elemento, sendo representados pela sintaxe de dois pontos duplos (::). Eles permitem intervir em fragmentos determinados do conteúdo, como a primeira linha ou a primeira letra de um parágrafo, sem a necessidade de criar novas tags no HTML.
