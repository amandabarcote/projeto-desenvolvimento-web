Incrementos no link

Os incrementos, ou atributos adicionais, são comandos que trazem melhor usabilidade e funcionalidade aos links (<a>). Exemplo:

O atributo target="_blank" serve para abrir a URL em uma nova guia quando o link é selecionado, enquanto o target="_self" (que é o padrão) serve para abrir o link na mesma guia em que o usuário já está.

O atributo download é utilizado para definir o nome do arquivo que será salvo no computador do usuário; esse nome será aplicado automaticamente assim que o download for iniciado.


Link Interno

    Existem dois tipos principais de links internos:


  • Um dos métodos é por meio de um link com âncora, que utiliza o atributo id em um elemento de destino e o símbolo # no atributo href do link para navegar até ele.

  • O segundo método para criar um link interno é através do atributo id, que funciona como um identificador único para um elemento, permitindo que o navegador role a página diretamente até ele.
