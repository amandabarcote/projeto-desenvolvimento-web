Imagem

    A imagem é um dos diversos itens que a linguagem de hipermídia nos permite adicionar ao documento HTML.


Marcação de imagem

    O atributo src define a localização da imagem, o alt fornece uma descrição caso a imagem não possa ser carregada, o height ajusta a altura e o width define a largura.


Ícone

    O atributo rel define o tipo de relação, o href indica o caminho do arquivo e o type especifica o tipo do arquivo, que, neste caso, é um ícone.

    No slide, também há alguns sites destacados em verde que permitem a criação gratuita de ícones.
