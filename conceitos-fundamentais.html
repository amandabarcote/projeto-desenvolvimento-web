HTML

    HTML - Hypertext Markup Language

    É uma linguagem de marcação de hipertexto baseada em tags, criada na década de 1990 por Tim Berners-Lee para a publicação de conteúdo em hipertexto, como textos, imagens, áudios e vídeos, que podem ser vinculados por meio de hiperlinks.


Tags

  • As tags definem para o navegador a marcação do conteúdo e possuem funcionalidades específicas. Sem o uso de tags, o navegador exibe apenas um texto plano.

    • Exemplo: <nome_tag> conteúdo </nome_tag>

  • As tags podem possuir atributos, ou seja, parâmetros que definem algum tipo de informação extra para o navegador.

    • Exemplo: <nome_tag atributo="valor"> conteúdo </nome_tag>


Estrutura básica do HTML

O texto deve ser inserido no <body>, enquanto as informações da página devem ser inseridas no <head>. Para melhorar a estrutura do site, utilizam-se tags; as mais simples são: <p>, <h1> e <br>.

A tag <p> é para parágrafos, enquanto a <h1> é para títulos. Para inserir um subtítulo, aumenta-se o número após o "h", como o <h2>, por exemplo. Já a tag <br> é utilizada para quebra de linha.


Marcação para formatação de texto

    As tags para a formatação de texto definem destaque, tamanho de fonte, cor e diversos outros estilos.

    Exemplo: <BIG> texto </BIG> e <font color="blue" size="14"> texto </font>
