Table

A tabela em HTML é estruturada pela tag <table>, que delimita seu início e fim, sendo composta pelos elementos <tr>, <th> e <td>. A tag <tr> define uma linha da tabela, a qual pode conter várias colunas; a tag <th> define as células de cabeçalho (títulos das colunas ou linhas) e a tag <td> armazena o conteúdo das células de dados. Além disso, existem a tag <caption>, que define um título descritivo para a tabela, e o atributo colspan, que determina quantas colunas uma informação ocupará horizontalmente na estrutura.

Sub

A tag <sub> é utilizada para configurar um texto como subscrito, renderizando-o com um tamanho de fonte reduzido e posicionado abaixo da linha de base convencional. Exemplo: x1 e x2 são as raízes possíveis numa equação do 2º grau. (o 1 e 2 estão nas tags sub).

Address

A tag <address> é utilizada para fornecer informações de contato do autor ou do proprietário de um documento ou artigo, podendo incluir dados como endereços de e-mail, URLs, endereços físicos, números de telefone ou links para redes sociais.

Cite

A tag <cite> é para citar o título de uma obra, como filmes, músicas, pinturas, livros, etc. O nome de uma pessoa não se enquadra no cite. Exemplo: O livro orgulho e preconceito é bem melhor que o filme.
